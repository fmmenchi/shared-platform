# i18n (agent)

- Copy is **colocated + typed**: `<name>.messages.ts` → `defineMessages('<ns>', { en/it/ar })`
  (type requires every locale + same keys). Read via `useMessages(catalog)` — typed `t`, base-locale
  fallback outside a provider.
- App overrides a label via the `messages` port, key `"<ns>.<key>"`. No central catalog, no i18n
  engine.
- `direction` derived from `Intl.Locale.maximize().script` (Baseline), never injected; use CSS
  logical properties.
